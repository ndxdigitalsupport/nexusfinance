SMS Features

NexusFinance uses Twilio for SMS messaging. Twilio supports Cambodia (+855) numbers and offers a $15 free trial.

API Endpoint

POST /api/sms/send — JWT-protected. Send an SMS to any phone number.

json{
  "to": "+85512345678",
  "text": "Your OTP code is 123456"
}

Sandbox Mode

No API keys needed for testing. When TWILIO_ACCOUNT_SID is unset, calls log to console.

Going Live

  1. Sign up at twilio.com (free $15 credit, no card needed)
  2. From Twilio Console → grab Account SID, Auth Token, and your trial phone number
  3. Set all 3 in .env — SMS goes live automatically
Pricing: $0.4418/SMS to Cambodia. $15 free trial credit — trial includes a free Twilio number, no purchase needed. Trial can only send to verified numbers.

4
Postman API Testing

Import the collection and test all 24 endpoints in minutes.

Setup

  1. Open Postman → Import
  2. Select postman/nexusfinance.postman_collection.json
  3. Set base_url to http://localhost:4001

Testing Flow

flowSession Login → JWT auto-saved → Test any endpoint
  1. Open Auth → Session Login, click Send
  2. JWT saves automatically to collection variables
  3. Browse any endpoint — Authorization header auto-injects
