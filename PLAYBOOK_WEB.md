# NexusFinance — Web & SMS Playbook

**v1.0** — Loan management platform: 3 portals, 24 API endpoints, Docker sandbox, Twilio SMS.

## Stack
React 19 · Tailwind v4 · Express · TypeScript · Supabase · Appwrite · Twilio · Docker

## Portals
- **Customer** — Apply, repay, track loans
- **Loan Officer** — Review, approve/reject/hold, KYC
- **Super Admin** — Users, config, audit logs, Telegram bot
